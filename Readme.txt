■　起動方法
「実行環境」フォルダ内のexeファイルをダブルクリックでゲームが起動します。

--------------------------------------------
■　終了方法
ESCキーを押すことで終了します。

--------------------------------------------
■　操作説明
（Logicool Gamepad 参照）
左スティック … 移動
Aボタン＋右スティック（上下） … チャージ
Bボタン … 攻撃
Aボタン … 決定

※ゲームパッド必須です。お手数おかけします。

--------------------------------------------
■　概要説明
4体の缶達が自分の炭酸で時間内に相手を沢山吹っ飛ばすパーティゲームです。
炭酸缶を振る動きをモチーフにした操作があり、みんなで楽しむ事が出来ます。
